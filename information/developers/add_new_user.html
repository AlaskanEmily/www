Adding a new user.

hydra is our master NIS server, with ender being our secondary slave server, for more details on how NIS was setup, see the file nis.debian.howto in /usr/doc/nis on hydra.

Get the new users /etc/passwd entry from mundook and then add it to /etc/passwd on hydra (using vipw to be safe).

Add the new user to group mercury in /etc/group on hydra (using vigr to be safe).

Then execute the following commands as root on hydra to update the NIS database.

cd /var/yp
make

Now add a directory on all the /home file systems that we have access to.
